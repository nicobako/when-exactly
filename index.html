Welcome to when-exactly

Work with dates and time the same way you think about dates and times.

Rationale

When we think about dates and times, we think in terms of years, months, weeks, days, hours, minutes, etc.

When-Exactly is a library that aims to bring these types into the hands of developers, so they can write more expressive and functional date/time code.

Overview

>>> import when_exactly as we

>>> year = we.year(2025)
>>> year.iso()
'2025'

>>> month = year.month(1)
>>> month.iso()
'2025-01'

>>> day = we.day(2025, 12, 25)
>>> day.iso()
'2025-12-25'

>>> day.month().iso()
'2025-12'

>>> year.months().iso()
'[2025-01,2025-02,2025-03,2025-04,2025-05,2025-06,2025-07,2025-08,2025-09,2025-10,2025-11,2025-12]'

>>> month.days()[0:-1:5].iso()
'[2025-01-01,2025-01-06,2025-01-11,2025-01-16,2025-01-21,2025-01-26]'
